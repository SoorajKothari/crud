# Introduction
 CRUD App 

# Stack
- RUST
- Postgres SQL DB
- Docker & Kubernates

# How to run

### To run on local (Only if you have postgres running on local)

Do `cargo run`

### How to run using docker compose (This will setup everything including DB)

Do `docker build . -t crud:latest`
Do `cd docker`
Do `docker compose up`
